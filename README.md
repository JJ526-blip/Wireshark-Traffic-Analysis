# Wireshark-Traffic-Analysis
Analysing different types of traffic and protocols in wireshark. 

Objective:
Capture live network traffic,
Analyze different protocols,
Identify normal vs suspicious activity,
Understand how data flows across a network

Tools Required:
Wireshark,
A computer with internet access

Project Tasks
1. Capture Network Traffic,
Open Wireshark,
Select your active network interface (Wi-Fi/Ethernet),
Start capturing packets,
Browse websites, 
2. Identify Protocols

Filter and analyze common protocols:

HTTP / HTTPS – web browsing
DNS – domain lookups
TCP / UDP – transport layer
ICMP – ping requests
